# 论述文结构

## 文章开头

### _阐释主题_

阐述主题+展开分析+亮明观点

### _事例导入_

一个或多个事例 + 展开分析 + 亮明观点

### _古今对比_

古代事例 + 现在事例 + 亮明观点

### _引经据典_

恰当引用 + 主题阐述分析 + 亮明观点

## 文章主体

### _理证_(讲道理)

#### basic

分论点 + 概述现状 + 分析原因/影响 + 总结

#### pro

basic + 解释分论点


### _例证_(举栗子)

分论点 + 过渡句 + 实例介绍 + 实例分析 + 总结

## 文章结尾

### _照应前文_

#### 标题

引入话题 + 过度 + 照应标题 + 深化主题

#### 开头

引入话题 + 过度 + 照应标题 + 深化主题

### _总结论点_

引入话题 + 总结论点 + 意义阐述

### _展望号召_

总结成绩 + 展望号召
